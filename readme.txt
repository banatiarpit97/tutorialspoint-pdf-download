1. Download all the files in this repository using the green button on top-right corner or git clone(preferred to create a separate folder for these files).
2. Make sure you have 'Node' installed on your machine to run the javascript file.(if not, install from https://nodejs.org/)
3. Open command prompt or terminal
4. Move to the directory in which you have downloaded the files.(use 'cd' command to navigate in command prompt/terminal).
5. Type the following command to initialize the resources needed for downloading tutorials(needs to be done only once):
	npm install
6. Type the following command to download the tutorial :
	node tutorialspoint --url="URL_OF_NOTES_PAGE_ON_TUTORIALSPOINT"

	URL_OF_NOTES_PAGE_ON_TUTORIALSPOINT should be replaced with the url of notes you want to download.
